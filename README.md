# Movie Watchlist App



## Features

- Search Movies – Search movies in real-time using the TMDB API  
- View Details – Get complete movie details including ratings, overview, and release date  
- Add/Remove Movies – Manage your personalized movie watchlist  
- Mark as Watched – Track which movies you have already seen  
- Local Storage Support – Watchlist and preferences persist even after page reload  
- Responsive UI – Works smoothly across desktop and mobile devices  

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- API: TMDB (The Movie Database)  
- Storage: Local Storage
